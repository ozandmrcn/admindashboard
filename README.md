# AdminDashboard

Welcome to the **AdminDashboard** project! A powerful, modern, and high-performance administration panel built with **Next.js 15**, **React 19**, and **Tailwind CSS v4**. This dashboard provides a comprehensive interface for managing products, tracking sales data through interactive charts, and monitoring system statistics.

## <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/Activities/Bullseye.png" alt="Bullseye" width="25" height="25" /> Project Overview

AdminDashboard enables administrators to:

- **Monitor Performance:** Real-time visualization of sales and category data using Chart.js.
- **Product Management:** Full CRUD (Create, Read, Update, Delete) operations for the product inventory.
- **Data Overview:** Quick access to key metrics like total users, orders, sales, and products.
- **Dynamic Content:** Interactive forms with real-time image previews.
- **Premium UI:** Fully responsive, modern, and professional interface designed with Tailwind CSS v4.
- **Mock Backend:** Integrated JSON Server for rapid development and testing.

## <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/Travel%20and%20places/Rocket.png" alt="Rocket" width="25" height="25" /> Features

- **Next.js 15 & React 19:** Utilizing the latest features like App Router, Turbopack, and enhanced Server/Client components.
- **Interactive Graphics:** Data visualization with **Chart.js** and **react-chartjs-2**.
- **Modern Styling:** Leverages the power of **Tailwind CSS v4** for high-performance, utility-first styling.
- **Mock API Service:** Uses **json-server** to simulate a RESTful backend with a `db.json` database.
- **State-of-the-art Forms:** Robust product management forms with dynamic fields and validation.
- **User Feedback:** Real-time notifications with **React Toastify**.
- **Iconography:** Clean and consistent visual language using **React Icons**.

To run the project locally, follow these steps:

```bash
# Clone the repository
git clone https://github.com/ozandmrcn/admindashboard.git

# Navigate to the project folder
cd admindashboard

# Install required dependencies
npm install

# In one terminal, run the mock API server
npm run server

# In another terminal, run the development server
npm run dev
```

### ⚙️ Environment Variables (.env Setup)

Create a `.env` file in the root directory and define the following variable to connect the frontend to the API:

```env
# API URL (Mock Server or Production API)
NEXT_PUBLIC_API_URL=http://localhost:4000
```

> ⚠️ **Note:** Ensure that the JSON Server is running on port 4000 (default) for the dashboard to fetch data correctly.
